What is CSS?

  1. CSS stands for Cascading Style Sheet
  2. CSS describes how HTML elements are to be displayed on screen, paper,or in other media.
  3. CSS is not neccesary CSS save a lot of work. It can control the layout of multiple web pages all at once.
  4. External style sheet are stared in CSS file

CSS can be added to HTML document in ways:

  • Inline- by using the style attribute inside HTML elementc
  • Internal- by using a style element in the head section.
  • Exterrnal - by using link element to link to an external CSS file.However in tutorial we will use inline and internal style, because this is easier to demonstrastrate, and easier for you to try it yourself.

When compairing CSS class vs ID, the different in that CSS class applied a style to multiple element. ID, on the other hand, applied applied a style to one unique element.ID is also special in that you can use a special URL to link directaly to an element and it's used by JS.

In CSS, selectors are used to target a specific element or range of elements on a web page. Once a element has been targeted, a style or a set of style can be applied to the element.

There is a wide range of selectors available. Two of the most common used are class and ID. Both are used to target elements to which a style should be applied.
